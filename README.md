## Simple-Upload

用到的东西：

* react

  UI 框架

* create-react-app

  react 脚手架

* react-app-rewired

  允许自定义 webpack 配置

* redux

  * redux

    状态管理工具

  * react-redux

    简化 redux 在 react 中的操作

  * redux-immutable

    替换原来的 combineReducers, 将 state 统一成 immutable 对象

  * redux-saga

    redux 中间件 将异步请求分离出来

* react-transition-group

  简化 css 动画的操作

* styled-components

  css-in-js 解决方案

* immutable

  Immutable persistent data collections for Javascript which increase efficiency and simplicity.
